## Features

With this app you can search within wikipedia

## Installation


1. download [/build/wikipedia.zip](/build/wikipedia.zip) 
2. copy in sdcard/downloads/ on your phone 
3. installl the app with the GerdaOs Installer App.


## How to use


- Select your language
- Soft-Left new search
- Soft-Right switch language
- Enter submit your search
